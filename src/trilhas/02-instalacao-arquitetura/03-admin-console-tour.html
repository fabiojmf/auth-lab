Tour pelo Admin Console

O Admin Console é seu centro de controle. Vamos explorar cada seção para que você saiba exatamente onde encontrar o que precisa.

O menu lateral

Clique em cada item para ver o que ele faz:

📱 Clients
📋 Client Scopes
🎭 Realm Roles
👤 Users
👥 Groups
🔗 Sessions
📊 Events
⚙️ Realm Settings
🔐 Authentication
🌐 Identity Providers
🗄️ User Federation

Aplicações conectadas a este realm. Cada app que usa o Keycloak para autenticação precisa ser registrada como um client.

Exemplos: sua SPA React, sua API backend, o próprio Admin Console

A barra superior

🔄 Realm Selector (esquerda)

  • Mostra o realm atual
  • Clique para trocar de realm
  • Botão "Create Realm" no dropdown
  • Tudo que você vê muda ao trocar

👤 Menu do Admin (direita)

  • Nome do usuário logado
  • Manage Account (self-service)
  • Sign Out
  • Realm info e server info

Padrão de navegação

O console segue sempre o mesmo padrão:

📋 Lista ex: todos os clients
→
📄 Detalhe ex: client "my-app"
→
📑 Abas Settings / Credentials / Roles

Lista / Detalhe / Abas. Esse padrão se repete em Clients, Users, Roles, Groups...

A URL conta a história

/admin/master/console/#/master/clients/abc123/settings
realm seção ID do recurso aba ativa

Olhando a URL você sempre sabe: em qual realm está, em qual seção, qual recurso e qual aba.

Pro tips

🔍

Ctrl+F funciona

Em listas longas de clients ou users, use a busca do navegador.

🖱️

Abrir em nova aba

Right-click / Open in new tab funciona na maioria dos links internos.

⌨️

Busca interna

Nas seções Users e Clients há um campo de busca no topo da lista.

🔖

Favoritar URLs

Salve nos favoritos as páginas que você acessa com frequência.
