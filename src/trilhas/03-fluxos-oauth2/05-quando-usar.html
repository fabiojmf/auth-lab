Quando usar cada fluxo

Responda as perguntas abaixo e descubra qual fluxo OAuth2 é o correto para o seu cenário:

Resumo rápido

Cenário Fluxo No Keycloak
App web com login de usuário Authorization Code Standard Flow = Sim
SPA (React/Angular) com login Authorization Code + PKCE Client public + Standard Flow
Backend chamando API (sem usuário) Client Credentials Service Account = Sim
CLI / ferramenta de dev Direct Grant Direct Grant = Sim
Sistema legado com tela própria Direct Grant Direct Grant = Sim
Renovar acesso sem novo login Refresh Token (automático com os outros fluxos)

💡 Regra de ouro

Tem humano logando? / Authorization Code (com PKCE se não tem backend)
Só máquina? / Client Credentials
Não pode fazer redirect? / Direct Grant (último recurso)
