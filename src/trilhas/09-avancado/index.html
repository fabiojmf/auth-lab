Avançado

Tópicos avançados para quem quer dominar o Keycloak além do básico. Customização, automação e arquiteturas complexas.

1

Custom Themes

Personalize login, emails e console com temas próprios sem alterar o código-fonte.
2

Custom SPIs

Estenda o Keycloak com providers customizados: autenticadores, user storage e event listeners.
3

API Admin (REST)

Automatize tudo que o Admin Console faz via API REST — users, clients, roles e mais.
4

Infrastructure as Code (Terraform)

Gerencie realms, clients e configurações como código versionado e reproduzível.
5

Multi-Tenancy

Estratégias para servir múltiplas organizações com uma única instância Keycloak.
6

Token Exchange

Troque tokens entre serviços — delegação, impersonation e downscoping em microservices.
