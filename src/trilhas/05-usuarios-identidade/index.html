Usuários e Identidade

De onde vêm os usuários? Como conectar bases externas, permitir login de outros provedores, e gerenciar identidades de forma centralizada.

1

Gerenciamento de Usuários

Criação, atributos, grupos e roles — tudo que você precisa saber sobre usuários locais no Keycloak.
2

User Federation

Conecte o Keycloak a bases externas como LDAP e Active Directory sem duplicar usuários.
3

Identity Providers (Brokering)

Delegue a autenticação para outro sistema — outro Keycloak, um IDP corporativo ou social.
4

Social Login

Google, GitHub, Facebook — configure login social com provedores pré-integrados.
5

Mapeamento e Fluxo de Identidade

Como atributos fluem de fontes externas até o token final — mappers, linking e first login flow.
